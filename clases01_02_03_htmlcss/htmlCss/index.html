HTML y CSS

HTML significa Lenguaje de Marcado de Hipertexto (Hypertext Markup Language) y sirve para hacer el esqueleto de una página web. Nos ayuda a maquetar la página. No es un lenguaje de programación.

CSS significa Hojas de Estilo en Cascada (Cascade Style Sheets) y es un lenguaje de estilización para la web. Sirve para darle las terminaciones y diferentes diseños a una página o sitio web.
